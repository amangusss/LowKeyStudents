# Low-Key Students — Frontend

Frontend часть проекта реализована с использованием HTML, CSS и Vanilla JavaScript. Основная задача — предоставление удобного интерфейса для взаимодействия пользователей с форумом.

## Описание функционала
- **Навигация**:
  - Главная страница со всеми постами
  - Страница пользователей
  - Профиль пользователя
- **Работа с постами**:
  - Создание, редактирование, удаление
  - Комментирование постов
- **Авторизация**:
  - Вход и регистрация через формы
  - JWT-токен для взаимодействия с Backend

## Установка и запуск
1. Откройте файл `index.html` в браузере.
